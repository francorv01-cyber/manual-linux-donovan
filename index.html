Manual de Comandos Linux

SORT

Ordena líneas de texto.

Opciones

  • -r
  • -n
  • -k

Ejemplos

sort archivo.txt
sort -r archivo.txt
sort -n numeros.txt
sort -k 2 archivo.txt

CUT

Extrae columnas de texto de una entrada.

Opciones

  • -d
  • -f

Ejemplos

cut -d ":" -f 1 /etc/passwd
cut -d "," -f 2 archivo.csv
